Interaction Design: Prototype


Since we've implemented our project on an actual Android device (a Droid X), We can't let you use that here. However, We made a demo video showing the prototype we have so far. It showcases the main screen of the app, especially the vertical sliders and the sliding drawer feature.


The code is fairly large, and that is mostly due to the highly complicated layout structure. We used RelativeLayouts to cut down on the drawing expenses. Also of note, the vertical sliders are not stock with the Android SDK. That entire visual widget had to be coded by hand.


As far as issues we're hitting so far go, code organization is the biggest. Getting the pop-up sliders to appear and be accessable requires a new Activity to be created, which makes the playing audio unacessable. The solution to this is known as a Service, but I've been having some trouble with the implementation. In the end however, I think we'll have a very cool working application. Video after the break.
